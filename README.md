# Termux Tutorial Website

A static tutorial website for Termux with beginner guide and advanced topics.

## File Structure

```
├── index.html          # Main tutorial page
├── beginner.html       # Beginner guide page
├── css/                # Stylesheets
├── js/                 # JavaScript files
├── data/               # JSON content
└── assets/             # Images and icons
```

## Quick Start

Serve the files with any static server:

```bash
python -m http.server 3000
```

## Deployment

The repository includes vercel.json for Vercel deployment. Also works with Netlify, GitHub Pages, or any static hosting.
